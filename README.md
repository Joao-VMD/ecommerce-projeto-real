# 🛒 Loja Virtual (Projeto Real em Desenvolvimento)

Projeto de desenvolvimento de uma loja virtual para um cliente real, com foco em criar uma plataforma de e-commerce funcional, moderna e escalável.

## 🚀 Sobre o projeto

Este projeto está sendo desenvolvido para atender a necessidade de um cliente real, com o objetivo de criar uma loja online com boa experiência do usuário, organização de produtos e estrutura preparada para crescimento.

**Atualização:** O projeto evoluiu! Após estruturar a interface base, avancei no desenvolvimento aplicando os primeiros scripts em **JavaScript** para trazer interatividade e preparar a aplicação para a lógica dinâmica do e-commerce. Durante todo o processo, venho aplicando boas práticas de front-end e organização de código limpo.

## 🚧 Status do projeto

Em desenvolvimento ativo (Avançando no Front-end dinâmico)

## ✨ Funcionalidades atuais

* Estrutura inicial da loja integrada
* Layout das páginas principais (`inicio.html`, `produtos.html`, `roupa.html`)
* Organização básica de produtos e folhas de estilo refinadas
* Navegação entre páginas
* **[Novo]** Implementação inicial de JavaScript para manipulação do DOM e lógica de interatividade

## 🔧 Próximas melhorias

* Evolução dos scripts JS para o sistema de carrinho de compras
* Sistema de login de usuários
* Responsividade aprimorada para dispositivos móveis
* Integração com backend e banco de dados
* Integração com métodos de pagamento

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)

## 💼 Contexto

Projeto desenvolvido para um cliente real, com foco em resolver uma necessidade prática de negócio, otimizar a experiência do usuário e melhorar a presença digital da marca através de código eficiente.

## ▶️ Como executar

1. Baixe ou clone o repositório.
2. Abra o arquivo `inicio.html` no navegador (recomendável utilizar a extensão *Live Server* no VS Code para renderização correta dos estilos e scripts).
